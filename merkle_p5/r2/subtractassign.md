
#### Nome
### -= (subtração e atribuição)

#### Exemplos

```pde
int a = 50; 
int b = 23; 
a -= b; // Atribui 27 à "a" 

```

#### Descrição

	
Combina subtração a atribuição A expressão**a -= b** é equivalente à expressão**a = a - b**.

#### Sintaxe
```pde
valor1 -= valor2
            
```
Parâmetros
valor1
int ou float
valor2
int ou float

#### Utilização

	
Web & Applicações
