#### Nome
### -= (subtração e atribuição)
<img height="25" src="../images/1pix.gif" width="1"/>

#### Exemplos

```pde
int a = 50; 
int b = 23; 
a -= b; // Atribui 27 à "a" 

```

#### Descrição

	
Combina subtração a atribuição A expressão**a -= b** é equivalente à expressão**a = a - b**.
<img height="25" src="../images/1pix.gif" width="1"/>
