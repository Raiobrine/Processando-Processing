
#### Nome
### exp()

#### Exemplos

```pde
float v1 = exp(1.0); 
println(v1);  // Imprime "2.7182817" 
 

```

#### Descrição
Retorna o número de Euler*e* (2.71828...) elevado a potência do parâmetro**valor.**

#### Sintaxe
```pde
exp(valor)

```
Parâmetros
value
int ou float: o expoente a elevar *e*<description>
</description>
