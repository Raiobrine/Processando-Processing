
#### Nome
### round()

#### Exemplos

```pde
float x = 9.2;  
int rx = round(x);  // Atribui 9 à rx
 
float y = 9.5; 
int ry = round(y);  // Atribui 10 à ry 
 
float z = 9.9; 
int rz = round(z);  // Atribui 10 à rz 
 

```

#### Descrição
Calcula o inteiro mais próximo ao valor do parâmetro. Por exemplo,**round(9.2)** retorna 9.

#### Sintaxe
```pde
round(valor)

```
Parâmetros
valor
float

#### Retorno

	
int
