#### Nome
### noCursor()
<img height="25" src="../images/1pix.gif" width="1"/>

#### Exemplos

```pde
// Press the mouse to hide the cursor 
void draw() 
{ 
  if(mousePressed == true) { 
    noCursor(); 
  } else { 
    cursor(HAND); 
  } 
} 

```

#### Descrição
Esconde o cursor de vista.  Não irá funcionar quando o programa for executado em um navegador web.
<img height="25" src="../images/1pix.gif" width="1"/>

#### Sintaxe
```pde
noCursor()

```
