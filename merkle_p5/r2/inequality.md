
#### Nome
### != (diferente)

#### Exemplos

```pde
int a = 22; 
int b = 23; 
if(a != b) { 
 println("a variável a não é igual a variável b"); 
} 

```

#### Descrição
Determina se uma
expressão não é equivalente a outra.

#### Sintaxe
```pde
valor1 != valor2
            
```
Parâmetros
valor1
int,
float, char, byte, boolean, String
valor2
int,
float, char, byte, boolean, String
