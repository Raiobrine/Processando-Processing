#### Nome
### != (diferente)
<img height="25" src="../images/1pix.gif" width="1"/>

#### Exemplos

```pde
int a = 22; 
int b = 23; 
if(a != b) { 
 println("a variável a não é igual a variável b"); 
} 

```
