
#### Nome
### break

#### Exemplos

```pde
char letter = 'B'; 
 
switch(letter) { 
  case 'A': 
    println("Alpha");  // Não é executado 
    break; 
  case 'B': 
    println("Bravo");  // Imprime "Bravo" 
    break; 
  default: 
    println("Zulu");   // Não é executado 
    break; 
} 

```

#### Descrição
Termina a execução de uma estrutura como  switch(), for(), or while(), e pula para o comando imediatamente seguinte

#### Sintaxe
```pde
break
            
```
