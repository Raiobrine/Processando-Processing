#### Nome
### floor()
<img height="25" src="../images/1pix.gif" width="1"/>

#### Exemplos

```pde
float x = 2.88;  
int a = floor(x);  // Atribui 2 à "a"

```

#### Descrição
Calcula o valor inteiro mais próximo ao valor do parâmetro**valor**,  menor ou igual a este.
