
#### Nome
### alpha()

#### Exemplos
<img border="0" height="100" src="media/alpha_.gif" width="100"/>

```pde
noStroke(); 
color c = color(0, 126, 255, 102); 
fill(c); 
rect(15, 15, 35, 70); 
float value = alpha(c);  //Sets "value" to "102" 
fill(value); 
rect(50, 15, 35, 70); 

```

#### Descrição
Extrai o valor alpha de uma cor

#### Sintaxe
```pde
alpha(cor)

```
Parâmetros
cor
qualquer valor do tipo de dados color

#### Retorno

	
float
