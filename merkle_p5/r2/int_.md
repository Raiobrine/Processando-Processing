#### Nome
### int()
<img height="25" src="../images/1pix.gif" width="1"/>

#### Exemplos

```pde
float f = 65.0; 
int i = int(f); 
println(f + " : " + i);  // Prints "65.0 : 65" 
 
char c = 'E'; 
i = int(c); 
println(c + " : " + i);  // Imprime "E : 69" 

```

#### Descrição
Converte um tipo de dados primitivo, string ou*array *na sua representação em número inteiro.
