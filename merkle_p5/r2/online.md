
#### Nome
### online

#### Exemplos

```pde
if(online) {  // of "if (online == true)" 
  ellipse(50, 50, 60, 60); 
} else { 
  line(0, 0, 100, 100); 
  line(100, 0, 0, 100); 
} 

```

#### Descrição
Confirma se um programa Processing está
executando em um navegador web. Esta variável é "true"
(n.t. verdadeira)  se o programa estiver*online *e "*fasle"* caso não esteja.

#### Sintaxe
```pde
online

```

#### Utilização

	
Web
