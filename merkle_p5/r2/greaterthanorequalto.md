
#### Nome
### >= (maior ou igual)

#### Exemplos

```pde
int a = 23; 
int b = 23; 
if(a >= b) { 
  println("variável a é maior ou igual a variável b"); 
} 

```



#### Descrição
Testa se o valor à
esquerda é maior que o valor à direita ou se os valores são equivalentes.

#### Sintaxe
```pde
valor1 >= valor2
            
```
Parâmetros
valor1
int ou float


valor2
int ou float
