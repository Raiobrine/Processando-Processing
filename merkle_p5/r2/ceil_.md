
#### Nome
### ceil()

#### Exemplos

```pde
float x = 8.22;  
int a = ceil(x);  // Atribui 9 à "a"

```

#### Descrição
Calcula o valor inteiro mais próximo que seja maior ou igual ao valor do parâmetro. Por exemplo,**ceil(9.03)** retorna o valor 10.

#### Sintaxe
```pde
ceil(valor)

```
Parâmetros
valor
float

#### Retorno

	
int
