#### Nome
### unhex()
<img height="25" src="../images/1pix.gif" width="1"/>

#### Exemplos

```pde
String hs = "FF006699"; 
int hi = unhex(hs); 
fill(hi); 
rect(30, 20, 55, 55); 
 

```

#### Descrição

Converte uma representação em String
de um número hexadecimal  em seu equivalente valor inteiro.
<img height="25" src="../images/1pix.gif" width="1"/>
