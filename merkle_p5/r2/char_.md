#### Nome
### char()
<img height="25" src="../images/1pix.gif" width="1"/>

#### Exemplos

```pde
int i = 65; 
char c = char(i); 
println(i + " : " + c);  // Imprime "65 : A" 
 
float f = 72.4; 
c = char(f); 
println(f + " : " + c);  // Imprime "72.4 : H" 
 
boolean b = true; 
c = char(b); 
println(b + " : " + c);  // Imprime "true : t" 

```

#### Descrição
Converte um tipo de dados primitivo, string ou*array *na sua representação em char
