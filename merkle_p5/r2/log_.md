#### Nome
### log()
<img height="25" src="../images/1pix.gif" width="1"/>

#### Exemplos

```pde
void setup() { 
  int i = 12; 
  println(log(i)); 
  println(log10(i)); 
} 
 
// Calculates the base-10 logarithm of a number 
float log10 (int x) { 
  return (log(x) / log(10)); 
} 

```

#### Descrição
Calcula o logaritmo natural (logaritmo na base*e*) de um número. Esta função espera que valoressejam maiores que 0.0.
