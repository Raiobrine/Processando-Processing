#### Nome
### <= (menor ou igual)
<img height="25" src="../images/1pix.gif" width="1"/>

#### Exemplos

```pde
int a = 22; 
int b = 23; 
if(a <= b) { 
  println("variável a é menor ou igual a variável b "); 
} 

```

#### Descrição
Testa se o valor à
esquerda é menor que o valor à direita ou se os valores são equivalentes.
<img height="25" src="../images/1pix.gif" width="1"/>

