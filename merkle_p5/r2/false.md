
#### Nome
### false

#### Exemplos

```pde
rect(30, 20, 50, 50); 
boolean b = false; 
if(b == false) { 
  line(20, 10, 90, 80);  // Esta linha é desenhada
} else { 
  line(20, 80, 90, 10);  // Esta linha não é desenhada
} 

```



#### Descrição
Palavra chace que representa o valor lógico "falso". O valor `falso ` pode ser atribuído apenas a variáveis do tipo **boolean** `. `

#### Sintaxe
```pde
false

```
