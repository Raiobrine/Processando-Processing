
#### Nome
### reverse()

#### Exemplos

```pde
String sa[] = { "OH ", "NY ", "MA ", "CA "}; 
sa = reverse(sa); 
print(sa);  // Imprime "CA MA NY OH " 

```



#### Descrição
Inverte a ordem de um array

#### Sintaxe
```pde
reverse(array)

```
Parâmetros
array
boolean[], byte[], char[], int[], float[], or String[]



#### Retorno

	
Array (do mesmo tipo da entrada)
