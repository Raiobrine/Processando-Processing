#### Nome
### reverse()
<img height="25" src="../images/1pix.gif" width="1"/>

#### Exemplos

```pde
String sa[] = { "OH ", "NY ", "MA ", "CA "}; 
sa = reverse(sa); 
print(sa);  // Imprime "CA MA NY OH " 

```
