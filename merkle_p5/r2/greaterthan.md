#### Nome
### > (maior)
<img height="25" src="../images/1pix.gif" width="1"/>

#### Exemplos

```pde
int a = 5; 
int b = 13; 
if(b > a) { 
 println("variable b is larger the variable a"); 
} 

```

#### Descrição
Testa se o valor à
esquerda é maior que o valor à direita.
