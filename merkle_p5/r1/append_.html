Nome  

append()
   
Exemplos   
String sa1[] = { "PR ", "SC ", "PI "}; 
String sa2[] = append(sa1, "AM "); 
print(sa2);  // Prints "PR SC PI AM"

Descrição   Expande um array em uma posição e adiciona o dado elemento à nova posição. O tipo de dados do parâmetro elemento deve ser do mesmo tipo do do array. 
   
Sintaxe   
append(array, elemento)
   
Parâmetros  
array   boolean[], byte[], char[], int[], float[], ou String[]
element   novos dados para o array
   
Retorno   Array (do mesmo tipo de dados da entrada)
   
Utilização   Web & Applicações
   
Relacionado   shorten()
expand()
