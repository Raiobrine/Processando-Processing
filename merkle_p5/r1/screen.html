Nome  

screen
   
Exemplos   
void setup() { 
  size(screen.width, screen.height); 
} 
 
void draw() { 
  line(0, 0, screen.width, screen.height); 
}

Descrição   Variável de sistema que armazena as dimensões da tela do computador. Por exemplo, se as dimensões da tela correntes são 1024x768, screen.width é 1024 e screen.height é 768. Estas dimensões são úteis na exportação de aplicações de tela inteira.  (n.t. screen se traduz com tela para o português do Brasil).
   
Sintaxe   
screen.width
screen.height
   
Utilização   Web & Applicações
