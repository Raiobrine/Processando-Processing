Nome  

&& (logical AND)
   
Exemplos   
for(int i=5; i<=95; i+=5) { 
  if((i > 35) && (i < 60)) { 
    stroke(0);    //Atribui a variável color a cor preta
  } else { 
    stroke(255);  //Atribui a variável color a cor branca
  } 
  line(30, i, 80, i); 
}
Descrição   Compara duas expressões e retorna verdadeiro (true) se ambas forem avaliadas como verdadeiras (true). Retorna false (false) se uma ou ambas forem avaliadas como falsas (false). A seguinte linstas mostra todas as possíveis combinações

true && false  // É avaliada como falsa porquê a segunda é falsa
false && true  // É avaliada como falsa porquê a primeira é falsa
true && true   // É avaliada como verdadeira porquê ambas são verdadeiras
false && false // É avaliada como falsa porquê ambas são falsas
   
Sintaxe   
expressão1 && expressão2
   
Parâmetros  
expressão1   qualquer expressão válida
expressão2   qualquer expressão válida
   
Utilização   Web & Applicações
   
Relacionado   || (lógico OU)
! (lógico NÃO)
