Nome  

+= (adição e atribuição)
   
Exemplos   
int a = 50; 
int b = 23; 
a += b; // Atribui 73 à "a"

Descrição   Combina adição à atribuição. A expressão  a += b é equivalente à a = a + b.
   
Sintaxe   
valor1 += valor2
   
Parâmetros  
valor1   int ou float
valor2   qualquer valor numérico do mesmo tipo de valor1 
   
Utilização   Web & Applicações
   
Relacionado   = (atribuição)
+ (adição)
-= (subtração e atribuição)
