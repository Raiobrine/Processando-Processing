Nome  

height (altura)
   
Exemplos   
noStroke(); 
background(0); 
rect(40, 0, 20, height); 
rect(60, 0, 20, height/2);
Descrição   A variável de sistema height armazena a altura da tela de visualização. Este valor é atribuído através do segundo parâmetro da função size(). Por exemplo, a chamada de função size(320,240)  atribui a variável height o valor de 240. O valor de height  é zero até que size() seja chamada. 
   
Sintaxe   
height
   
Utilização   Web & Applicações
   
Relacionado   width (largura)
