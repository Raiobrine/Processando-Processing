Nome  

!= (diferente)
   
Exemplos   
int a = 22; 
int b = 23; 
if(a != b) { 
 println("a variável a não é igual a variável b"); 
}

Descrição   Determina se uma expressão não é equivalente a outra.
   
Sintaxe   
valor1 != valor2
   
Parâmetros  
valor1   int, float, char, byte, boolean, String
valor2   int, float, char, byte, boolean, String
   
Utilização   Web & Applicações
   
Relacionado  
 ==  igual
 >_   maior
 >=  maior ou igual
 <  menor
 <=  menor ou igual
