openStream()

Descrição   Método simplificado para se abrir uma InputStream em Java. Esta função é útil quando se quer abrir facilmente arquivos da pasta de dados ou de um URL, mas se quer um objeto InputStream de modo que se possa utilizar outros métodos Java para se term mais controle de como a strem é lida.
   
Sintaxe   
openStream(arquivo)
   
Parâmetros  
arquivo   String: nome de um arquivo na pasta de dados (data), URl, ou outro arquivo local e sua localização.
   
Retorno   byte[]
   
Utilização   Web & Applicações

loadStrings()
