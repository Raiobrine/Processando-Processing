Nome  

; (semicolon)

   
Exemplos   
int a;         // Declaration statement 
a = 30;        // Assignment statement 
println(i);  // Function statement

Descrição   A statement terminator which separates elements of the program. A statement is a complete instruction to the computer and the semicolon is used to separate instructions (this is similar to the period "." in written English). Semicolons are also used to separate the different elements of a for() structure.
   
Sintaxe   
statement;
   
Parâmetros  
statement   a single statement to execute
   
Utilização   Web & Applicações
   
Relacionado   for()
