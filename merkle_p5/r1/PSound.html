Nome  

PSound
   
Exemplos   
PSound soundA; 
 
void setup() { 
  soundA = loadSound("clong.wav"); 
  soundA.loop(); 
  framerate(24); 
} 
 
void draw() { 
  println(soundA.time()); 
}

Descrição   Classe para armazenar e tocar sons no formato .wav.
   
Métodos  
play()
   Tocar o som uma vez
loop()
   Tocar o som continuamente
noLoop()
   Parar de tocar o som continuamente
pause()
   Pausar a reprodução do som em playback
stop()
   Parar a reprodução do som em playback
volume()
   Ajusta o volume da reprodução do som em playback
duration()
   Retorna a duração do som em segundos
time()
   Retorna o momento atual de reprodução do som em playback
   
   
Construtores   
PSound(file)
   
Parâmetros  
file   String: nome do arquivo de som a carregar
   
Utilização   Web & Applicações
