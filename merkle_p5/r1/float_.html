Nome  

float()
   
Exemplos   
int i = 65; 
float f = float(i); 
println(i + " : " + f);  // Imprime "65 : 65.0" 
 
char c = 'E'; 
f = float(c); 
println(c + " : " + f);  // Imprime "E : 69.0"

Descrição   Converte um tipo de dados primitivo, string ou array na sua representação em ponto-flutuant
   
Sintaxe   
float(valor)
   
Parâmetros  
valor   int, char, byte, boolean, String, int[], char[], byte[], boolean[], String[]
   
Retorno   float
   
Utilização   Web & Applicações
