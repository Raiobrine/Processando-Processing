Nome  

void
   
Exemplos   
void setup() { 
  size(200, 200); 
} 
 
void draw() { 
  line(10, 100, 190, 100); 
}

Descrição   Palavra chave  utilizada para indicar que uma função não retorna valor. Cada função deve ou retornar um valor de um tipo específico de dados ou especificar que não retorna nada através da palavra chave Bvoid. 
 
Sintaxe   
void funcao {
  comandos
}
   
Parâmetros  
funcao   qualquer função que está sendo definida ou implementada
comandos   quaisquer comandos válidos
   
Utilização   Web & Applicações
