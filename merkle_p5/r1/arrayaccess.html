Nome  

[] (acesso a vetor)
   
Exemplos   
int[] numeros = new int[3]; 
numeros[0] = 90; 
numeros[1] = 150; 
numeros[2] = 30; 
int a = numeros[0] + numeros[1]; // Atribui 240 à variável a
int b = numeros[1] + numeros[2]; // Atribui 180 à variável b

Descrição   O operador de acessa a vetor é utilizado para especificar a localização de um elemento em um vetor. O dado nesta localização pode ser definido pela sintaxe vetor[elemento]  ou lido com a sintaxe valor =  vetor[elemento].
   
Sintaxe   
tipodedado[]
vetor[element]
   
Parâmetros  
tipodedado   qualquer tipo de dado primitivo, incluindo classes definidas pelos usuários
vetor   qualquer nome válido de variável
elemento   int: não pode exceder o comprimento do vetor-1
   
Utilização   Web & Applicações
   
Relacionado   Array
