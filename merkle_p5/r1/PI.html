PI (3.14159...)
   
Exemplos   
float f = 0.0; 
beginShape(POLYGON); 
while(f < PI) { 
  vertex(width/2 + cos(f)*40, height/2 + sin(f)*40); 
  f += PI/12.0; 
} 
endShape();
Descrição   PI é uma constante matemática de valor constante igual à 3.14159265358979323846. É a razão entre a circunferência de um círculo e seu diâmetro. É útil em combinações com as funções trigonométricas sin() e cos().
