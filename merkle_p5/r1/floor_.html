Nome  

floor()
   
Exemplos   
float x = 2.88;  
int a = floor(x);  // Atribui 2 à "a"

Descrição   Calcula o valor inteiro mais próximo ao valor do parâmetro valor,  menor ou igual a este.
   
Sintaxe   
floor(valor)
   
Parâmetros  
valor   float
   
Retorno   int
   
Utilização   Web & Applicações
   
Relacionado   ceil()
round()
