Nome  

saveBytes()
   
Exemplos   
byte[] nums = { 0, 34, 5, 127, 52}; 
 
// agora escreva os byes em um arquivo
saveBytes("numbers.txt", nums);

Descrição   Oposta a loadBytes(), escreverá um array de bytes em um arquivo. Este arquivo é salvo na pasta de esboços, a qual pode ser aberta ao selecionar "Show sketch folder" do  menu  "Sketch".
   
Sintaxe   
saveBytes(filename, bytes)
   
Parâmetros  
filename   nome do arquivo a escrever
bytes   array de bytes a escrever
   
Retorno   Nenhum
   
Utilização   Application
   
Relacionado   loadStrings()
loadBytes()
saveStrings()
