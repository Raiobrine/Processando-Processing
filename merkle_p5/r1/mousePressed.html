Nome  

mousePressed
   
Exemplos   
// clique na imagem para mudar o valor
// do retângulo
void draw() { 
  if (mousePressed == true) { 
    fill(0); 
  } else { 
    fill(255); 
  } 
  rect(25, 25, 50, 50); 
}

Descrição   Variável que armazena se um botão do mouse está pressionando. O valor da variável de sistema mousePressed é true se um botão do mouse está pressionado e false se um botão nao está pressionado. 
   
Utilização   Web & Applicações
