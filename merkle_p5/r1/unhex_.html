Nome  

unhex()
   
Exemplos   
String hs = "FF006699"; 
int hi = unhex(hs); 
fill(hi); 
rect(30, 20, 55, 55);

Descrição  
Converte uma representação em String de um número hexadecimal  em seu equivalente valor inteiro. 
   
Sintaxe   
unhex(str)
   
Parâmetros  
str   String
   
Retorno   int
   
Utilização   Web & Applicações
   
Relacionado   hex()
binary()
unbinary()
