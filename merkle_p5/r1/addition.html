Nome  

+ (addição)
   
Exemplos   
int a = 50 + 5;  // Atribui 55 à a
int b = a + 5;   // Atribui 60 à b
 

String s1 = "Chernenko"; 
String s2 = "Brezhnev"; 
String sc1 = s1 + s2; 
String sc2 = s1 + ", Andropov, " + s2; 
println(sc1);  // Prints ChernenkoBrezhnev 
println(sc2);  // Prints Chernenko, Andropov, Brezhnev 

String s1 = "Gorbachev"; 
int i = 1987; 
String sc1 = s1 + i; 
println(sc1);  // Prints Gorbachev1987

Descrição   Adiciona dois valores ou concatena duas strings. Como operador matemático, calcula a soma de duas variáveis. Como operador de string, combina duas strings em uma única, e caso necessário faz antes a conversão de tipos de dados primiticos em String. 
   
Sintaxe   
valor1 + valor2
   
Parâmetros  
valor1   String, int, float, char, byte, boolean
valor2   String, int, float, char, byte, boolean
   
Utilização   Web & Applicações
   
Relacionado   ++ (incremento)
+= (atribuição e soma)
- (subtração)
