Nome  

trim()
   
Exemplos   
String s = "  Somerville MA "; 
println(s);  // Imprime "  Somerville MA " 
String s2 = trim(s); 
println(s2); // Imprime "Somerville MA"

Descrição   Remode os caracteres em branco do início e do fim de uma String. Adicionalmente aos padrões de caracteres em brnco, como espaço,  carriage return, tab, esta função também remoce o caracter Unicode "nbsp".
   
Sintaxe   
trim(str)
   
Parâmetros  
str   qualquer String
   
Retorno   String
   
Utilização   Web & Applicações
