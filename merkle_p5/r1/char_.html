Nome  

char()
   
Exemplos   
int i = 65; 
char c = char(i); 
println(i + " : " + c);  // Imprime "65 : A" 
 
float f = 72.4; 
c = char(f); 
println(f + " : " + c);  // Imprime "72.4 : H" 
 
boolean b = true; 
c = char(b); 
println(b + " : " + c);  // Imprime "true : t"

Descrição   Converte um tipo de dados primitivo, string ou array na sua representação em char
   
Sintaxe   
char(valor)
   
Parâmetros  
valor   int, float, byte, boolean, String, int[], float[], byte[], boolean[], String[]
   
Retorno   char
   
Utilização   Web & Applicações
