Nome  

boolean
   
Exemplos   
boolean a = false; 
if (!a) { 
  rect(30, 20, 50, 50); 
} 
a = true; 
if (a) { 
  line(20, 10, 90, 80); 
  line(20, 80, 90, 10); 
}
Descrição   Boolean é o tipo de dados prádefinido para os valores true e  false (n.t. verdadeiro e falso). É comum utilizar valores boolean em comandos de controle que determinam o fluxo de execução de um programa. Na primeira vez que uma variável é escrita, ela deve ser declarada com um comando que expressa seu tipo de dados.
   
Sintaxe   
boolean var
boolean var = valorbooleano
   
Parâmetros  
var   nome da variável que fará referência ao valor
valorbooleano   true ou false (verdadeiro ou falso, respectivamente)
   
Utilização   Web & Applicações
   
Relacionado   true (verdadeiro)
false (falso)
