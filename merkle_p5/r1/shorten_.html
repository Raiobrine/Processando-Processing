Nome  

shorten()
   
Exemplos   
String sa1[] = { "OH ", "NY ", "CA "}; 
String sa2[] = shorten(sa1); 
print(sa1);  // Imprime "OH NY " 
println(); 
sa2 = shorten(sa2); 
print(sa2);  // Imprime "OH "

Descrição   Decrementa o tamanho de um array em um elemento e retorna o array menor.
   
Sintaxe   
shorten(array)
   
Parâmetros  
array   boolean[], byte[], char[], int[], float[], or String[]
   
Retorno   Array (do mesmo tipo de dados da entrada)
   
Utilização   Web & Applicações
   
Relacionado   append()
expand()
