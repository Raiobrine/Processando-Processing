Nome  

, (vírgula)
   
Exemplos   
// Uso da vírgula para separar uma lista de declarações de variáveis
int a=20, b=30, c=80; 
 
// Uso da vírgula para separar uma lista de atribuições a um vetor
int[] d = { 20, 60, 80 }; 
 
// Uso da vírgula para separar uma lista de parâmetros 
// passados a uma função
line(a, b, c, b); 
line(d[0], d[1], d[2], d[1]);

Descrição   Separa parâmetros em chamadas de função em elementos em atribuições
   
Sintaxe   
valor, ..., valorN
   
Parâmetros  
valor1, ..., valorN   qualquer int, float, byte, boolean, color, char, String
   
Utilização   Web & Applicações
