Nome  

<= (menor ou igual)
   
Exemplos   
int a = 22; 
int b = 23; 
if(a <= b) { 
  println("variável a é menor ou igual a variável b "); 
}

Descrição   Testa se o valor à esquerda é menor que o valor à direita ou se os valores são equivalentes.
   
Sintaxe   
valor1 <= valor2
   
Parâmetros  
valor1   int ou float
valor2   int ou float
   
Utilização  
   
Relacionado  
 >_   maior
 >=  maior ou igual
 <  menor
 ==  igual
 !=  diferente
