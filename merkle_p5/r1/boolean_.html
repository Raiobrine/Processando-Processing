Nome  

boolean()
   
Exemplos   
float f = 0.0; 
boolean b = boolean(f); 
println(f + " : " + b);  // Imprime "0.0 : false" 
 
int i = 1; 
b = boolean(i); 
println(i + " : " + b);  // Imprime "1 : true" 
 
i = 3445; 
b = boolean(i); 
println(i + " : " + b);  // Imprime "3445 : true"

Descrição   Converte um tipo de dados primitivo, string ou array na sua representação booleana. O número o é convertido para false (n.t. falso) e todos os demais números como true (n.t. verdadeiro).
   
Sintaxe   
boolean(valor)
   
Parâmetros  
valor   int, float, char, byte, String, int[], float[], char[], byte[], String[]
   
Retorno   boolean
   
Utilização   Web & Applicações
   
Relacionado   boolean
